# ROUND Utilities Guide

This directory contains specialized tools for inspecting and exporting the raw Phasic Data generated by the ROUND benchmark suite.

## 🛠️ Tools

### 1. Phasic Inspector (`inspect_pt.py`)

A lightweight utility for auditing the structure of `.pt` result files. Use this to verify that a benchmark run captured the correct dimensions (e.g., hidden states, success rates, or coordinates).

**Usage:**

```bash
python inspect_pt.py <path_to_results.pt>
```

**Output Includes:**

* Dictionary Keys
* Tensor/Array Shapes
* Scalar Metadata (neurons, learning rate, results)

---

### 2. Phasic Markdown Exporter (`export_pt_to_md.py`)

Converts binary `.pt` experiment data into a human-readable, high-fidelity Markdown report. This is optimized for the **"Diamond Lock"** verification process.

**Usage:**

```bash
python export_pt_to_md.py <path_to_results.pt>
```

**Output Sections:**

* **Metadata:** Essential parameters for the run.
* **Bit Persistence Grid:** A text-based representation of the 256x8 bit-reconstruction heatmap. Shows precisely where the model achieved "Green Wall" success vs where it "Lost" bits.
* **Phasic Manifold Coordinates:** Groups the 512D -> 2D Hypertorus projection data by Sequence ID and Step for easier mapping analysis.

## 🛰️ Integration

These utilities are designed to be called manually for deep-dive diagnostics or as part of an automated reporting pipeline following a `UIT_run_battery_targeted.py` execution.
